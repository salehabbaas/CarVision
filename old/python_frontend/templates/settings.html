System Settings

Global configuration for the CarVision detection engine and interface

🧠Model Overview

Auto: Runs YOLO first, then falls back to Contour when YOLO is not confident.
YOLO: Deep learning detector (best accuracy) — requires models/plate.pt.
Contour: Fast traditional OpenCV pipeline — works with no model file.
OCR: EasyOCR reads text from detected regions; language list is configurable.

🤖Detection Engine

Cameras set to "inherit" will use this mode. YOLO requires a trained model file at models/plate.pt
💡 YOLO mode is more accurate but requires placing a plate.pt model file in the models/ directory. Contour mode works out-of-the-box with no model file needed.
